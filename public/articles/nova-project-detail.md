# Nova 模块化作品集系统

**Nova** 不仅仅是一个博客模板，它是一个高度可配置的个人品牌展示资产。

## 核心功能
- **双流架构**：智能区分“技术文章”与“工程项目”，适应开发者多维度的展示需求。
- **管理看板**：内置受保护的后台（密码验证），支持实时修改本地数据存储（LocalStorage）。
- **动态资源加载**：通过 fetch API 异步获取 `public/articles/` 目录下的 `.md` 文件。
- **全平台适配**：从超宽显示器到移动端手势操作，均经过精细调优。

## 技术挑战：Markdown 渲染与代码高亮
在项目中，我们使用了 `marked` 库来处理 Markdown。为了保证样式的一致性，我们在 Tailwind 中配置了专门的 `prose` 类，并针对代码块和引用进行了定制化设计。

## 为什么它与众不同？
传统的 CMS 系统太重，而纯静态生成（如 Hexo）在内容调整上缺乏灵活性。Nova 走了一条中间道路：**配置驱动架构**。你可以直接在运行时的控制台中点击修改项目权重，并立即在主页看到变化。
