# 基于 VISSIM 的城市路网数字孪生研究

这是一个结合了交通工程专业知识与前端可视化技术的探索性项目。

## 项目愿景
传统的交通仿真软件往往界面陈旧，难以向非专业决策者展示复杂的流体力学现象。本项目旨在利用 **Web 技术** 重新定义仿真结果的交互方式。

## 技术实现
1. **数据提取**：使用 Python 脚本解析 VISSIM 的 fzp 车辆轨迹文件。
2. **逻辑建模**：计算路段流量、密度及延误等核心交通流指标。
3. **前端渲染**：通过 Canvas 或 D3.js 将静态数据转化为动态的时空分布图。

## 成果展示
通过该系统，我们可以清晰地观察到由于信号配时失误导致的“冲击波”回传现象，并为优化策略提供直观的支撑数据。
